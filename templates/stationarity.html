Stationarity Tool

Stationary data is simply data whose statistical measures such as mean, variance and standard deviation are constant. If stock data is stationary we can attempt to predict future values in a variety of ways e.g. linear regression, using the mean or more complex machine learning models. This tool performs an adf test on the entered stock and returns the adf values

P-Value ADF Statistic Stationary
Less Than 0.05 Less Than All Critical Values Stationary
Less Than 0.05 Not less than all Critical Values Stationary
Greater than 0.05 Less than all Critical Values Stationary
Greater than 0.05 Not less than all Critical Values Non Stationary
